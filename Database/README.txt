To initialize the database:

1. Run the Script.sql using 'sa' login.
The Script creates a new database 'LocusnineDB', a new login 'locuser' and a table 'tblUsers'.

OR

2. Restore the Locusnine.bak file.
Please Note : The backup file is generated in version 14.

Create login 'locuser' with password 'locusnine' and assign LocusnineDB
to the login.
